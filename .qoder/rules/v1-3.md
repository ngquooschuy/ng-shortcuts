# 1. formatShortcut Utility

## Public API

```ts id="vt58cm"
hotkeys.format('mod+k');
formatShortcut('mod+k');
```

## Output Rules

### Mac

```text id="2l4ow0"
⌘K
⌥1
⇧↵
⌘⇧P
```

### Windows / Linux

```text id="df35lz"
Ctrl+K
Alt+1
Shift+Enter
Ctrl+Shift+P
```

## Support

* mod
* ctrl
* meta
* alt
* shift
* enter
* esc
* tab
* arrows
* sequences

Example:

```ts id="4r6qcz"
formatShortcut('g h')
```

Output:

```text id="x2v6t1"
G H
```

# 3. HotkeysCheatsheetComponent

Standalone Angular component.

## Selector

```html id="6g9p4t"
<ngx-hotkeys-cheatsheet />
```

## Usage

```html id="aqx1xz"
<ngx-hotkeys-cheatsheet
  title="Keyboard Shortcuts">
</ngx-hotkeys-cheatsheet>
```

## With Manual Items

```html id="s7y9u0"
<ngx-hotkeys-cheatsheet
  [items]="[
    { keys: 'mod+k', label: 'Search' },
    { keys: 'mod+s', label: 'Save' },
    { keys: 'g h', label: 'Go Home' }
  ]">
</ngx-hotkeys-cheatsheet>
```

## UI Layout

```text id="t7m9s2"
Keyboard Shortcuts

Search .......... ⌘K
Save ............ ⌘S
Go Home ......... G H
Close ........... Esc
```

## Requirements

* responsive
* keyboard accessible
* semantic HTML
* clean layout
* no CSS frameworks

# 4. Optional Registry Metadata

Allow registering descriptions:

```ts id="n8l7h0"
hotkeys.on('mod+k', openSearch, {
  description: 'Search'
});
```

Cheatsheet can optionally read registered shortcuts automatically.

If too complex, support manual items first.
